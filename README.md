# 🚀 Portfólio Profissional - Davi Henrick

Este projeto é um portfólio moderno, responsiva e clean, desenvolvido para destacar as experiências e habilidades de **Davi Henrick**.

---

## 🔗 Link do Projeto
Você pode acessar o portfólio online aqui: 
👉 [**Visualizar Portfólio**](https://davihenrick-analytics.vercel.app)

---

## 🛠️ Tecnologias de Desenvolvimento
Para garantir performance e um design elegante, utilizei:
* **HTML5** (Estruturação semântica)
* **CSS3** (Estilização avançada com Flexbox/Grid)
* **JavaScript** (Interatividade e animações)

## 📌 Seções do Site
- **Trajetória:** Resumo das experiências profissionais passadas.
- **Projetos:** Galeria com os principais trabalhos e soluções.
- **Habilidades:** Hard skills técnicas em destaque.
- **Contato:** Links diretos para redes profissionais.

## 👩‍💻 Sobre o Desenvolvimento
Este projeto foi desenvolvido com carinho por **Vânia Melo**. O objetivo foi criar uma interface que transmita profissionalismo e facilite a leitura por parte de recrutadores e parceiros de negócios.
